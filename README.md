# AlohaInk by Addy Davis

Beautiful henna tattoos by Addy Davis — Marin County, CA.

## How to publish on GitHub Pages

1. Create a free account at [github.com](https://github.com) if you don't have one
2. Click the **+** button (top right) → **New repository**
3. Name it exactly: `alohaink` (or your GitHub username + `.github.io` for a root site)
4. Set it to **Public**
5. Click **Create repository**
6. On the next screen, click **uploading an existing file**
7. Drag and drop **both files** (`index.html` and `README.md`) into the upload area
8. Click **Commit changes**
9. Go to **Settings** → **Pages** (left sidebar)
10. Under "Branch", select `main` and click **Save**
11. Your site will be live in ~1 minute at:
    `https://YOUR-USERNAME.github.io/alohaink`

## Adding photos later

Open `index.html` in any text editor — or just use the Portfolio page's
upload button directly on the live site. Photos uploaded via the site
are saved in your browser automatically.
